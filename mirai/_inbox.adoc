### 文字起こし整形

```text
文字起こしデータを、自然な文章単位に整形してください。

【目的】
- タイムスタンプ付きの文字列を、文単位で区切り、読みやすく整理する。
- 一文ごとに1オブジェクトとし、タイムスタンプをキー、文を値とするJSON形式で出力する。
- 元の文体・語順は保持し、口語調（例：「〜ですね」「〜だよ」など）も維持する。
- **内容を省略せず**、意味が自然に伝わるよう調整する。

【文の区切りルール】
- 「。」「？」「！」などの句点で区切る。
- ただし、文の長さが極端に短い場合（10文字未満など）は、前後と結合し、1文としてまとめる。
- 会話の途中で不自然に分断されないよう注意する。

【タイムスタンプの扱い】
- 正確であることが重要。
- 各文に、その文の冒頭に近い元のタイムスタンプを割り当てる。
- 分割によって開始時刻が不明な文には、前後のタイムスタンプ・文の位置・文字数の比率をもとに補完する。
- 各文のタイムスタンプはユニークであること（重複不可）。
- フォーマットは `hh:mm:ss`（ゼロ埋め、24時間表記）。

【出力形式】
- JSONオブジェクトで出力してください。例：
{
  "00:00:03": "こんにちは、みなさん。",
  "00:00:07": "今日はAIの話をします。",
  ...
}

【出力制御】
- 元データが長文の場合、出力しきれない可能性があるので、適切な長さごとに 段階的に分けて出力 してください。
- 次の出力が必要な場合、ユーザーの指示を待ってから続きを出力してください。

【元データ】
添付ファイルを参照してください。


タイムスタンプ付き文字列データを、結合・分割し整形してください。
【目的】
- 一文ごとに1行（1オブジェクト）にする。
- 意味の区切れ（句点「。」「？」「！」など）を基準に文を区切る。
- 元の語順・文体は保ち、自然な文章として読めることを優先する。ただし、省略はしない。
- 会話文としての口調（語尾の「ね」「よ」など）も極力維持する。
- タイムスタンプ
  - **その文の開始位置に最も近いタイムスタンプ**を割り当てて残す。
  - 分を区切ることで割り当てるタイムスタンプがない場合は、前後のタイムスタンプと文字数などから推測し割り当てる。
  - タイムスタンプは重複できない。
【出力形式】
JSON形式で、以下のようにしてください：
{
  "hh:mm:ss": "一文目。",
  "hh:mm:ss": "二文目。",
  ...
}

【元データ】
```

### チャプター生成

```text
動画の視聴用チャプターを作成してください。
以下の条件をすべて守ってください。
- 各チャプターの長さは10秒以上にすること
- チャプターのタイトルは短く、端的に書くこと
- タイトルには記号・絵文字・装飾を使わないこと
- 「チーム未来」は「チームみらい」と表記すること
- 「安野貴博」は「安野党首」と表記すること
- 「高山聡史」は「高山党幹事長」と表記すること
- 「須田英太郎」は「須田氏」と表記すること
- 「ながた町」は「永田町」と表記すること
- 出力形式は次のJSONのようにしてください（例）：
{
  "00:00:10": "冒頭のあいさつ",
  "00:02:30": "チームみらいの紹介"
}
- 出力は結果のみを表示すること（コメントや説明、囲み・マークなど一切なし）
- ユーザーがコピーしやすいように、余計な文は一切含めないこと
```

動画タイトルは誹謗中傷だが、動画内ではSNS批判に対して語っている。
動画の内容を、超ざっくりまとめると
- 精神的ダメージのポイントは「質」ではなく「量」
- 現状では、SNS批判の「量」に対抗する法律がない
- 今後、安全保障上の理由からSNS実名化が進み解決すると予想
- SNS実名化は過去に遡って調査される可能性が高い
- 現状において、アマチュアスポーツでは収入と批判のバランスが悪い

個人的に考えたこと
アマチュアスポーツ選手（約400万円〜650万円）
アマチュアのトップレベルのスケート選手（フィギュアスケートやスピードスケート）では、平均年収が1,000万円から3,000万円程度

- 自身のコメントは、実名で発信できる内容か？
  - 発信する場所に応じた丁寧さ
- 自身のコメントは、束となった時に誹謗中傷とならないか？
- 政治家のリスクとリターンのバランス
  - チームみらいの心が折れたときの国損


動画の内容を、もう少しだけ詳しくまとめると
- 問題の現状
  - 選手へのSNS批判が殺到することがあり、批判の「質」ではなく「量」が選手を精神的に傷つけている。
  - 現行法では、批判の「量」に対抗する法律がない。
  - プロスポーツの批判とは異なり、収入と批判のバランスが悪い。
  - 「誹謗中傷をやめよう」という啓蒙活動は強制力がなく、実効性に乏しい。
- 根本的な解決策：SNSの「実名化」
  - 最終的には、全てのSNSが実名化されると予想されており、過去に遡って実質的に身元が特定されるようになると考えている。
  - これは、匿名SNSが選挙介入や世論操作に利用され、安全保障上の脅威となりうるため、政府が匿名アカウントの調査を進めるだろうという見解に基づく。
  - 実名化が進めば無責任な発言が減り、結果的に誹謗中傷は大幅に解決されると推測している。
- 具体的な対策とアスリートへのアドバイス
  - 誹謗中傷問題は「啓蒙」ではなく、「強制」のフェーズに移行したほうがよい。
  - 選手個人に任せるのではなく、競技団体や選手会が弁護団を結成し、選手に代わって誹謗中傷を特定し提訴すべき。
  - アスリートはメンタルヘルス維持のため、SNSは情報発信に限り利用し、それ以外は控える。人生の幸福度や競技力向上に、SNS批判を受け止めるのはマイナスである。
